# LyricSync
a Lyric Presentation Software to help Churches to sync their lyrics with their church music

to Start the Frontend 
```
bun dev
```
to start the backend
```
cd backend && bun dev
```

stacks
- React
- TypeScript
- Bun
- Vite
- Tailwind
- Drizzle
- Socket.io
- Sqlite
- Multer
- Express
